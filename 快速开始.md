# 🚀 CQL批量转换 - 快速开始

## 选择你的方案并执行：

### ⚡ 方案A: PowerShell + 在线API (最简单)
```powershell
# 直接运行，无需安装任何软件
.\convert_cql_to_elm.ps1
```
✅ 优点: 零配置，立即可用  
⚠️ 缺点: 需要网络，速度较慢 (预计20-30分钟)

---

### 🔥 方案B: PowerShell + 本地Java工具 (推荐)
```powershell
# 1. 检查Java (需要Java 11+)
java -version

# 2. 如果没有Java，下载安装: https://adoptium.net/

# 3. 自动下载转换器并执行
.\convert_cql_to_elm_local.ps1 -DownloadTranslator
```
✅ 优点: 速度最快 (预计2-5分钟)，离线可用  
⚠️ 缺点: 需要安装Java

---

### 💻 方案C: Node.js (开发者推荐)
```powershell
# 1. 检查Node.js
node --version

# 2. 如果没有Node.js，下载安装: https://nodejs.org/

# 3. 运行转换
node batch_convert_cql.js
```
✅ 优点: 跨平台，易集成 (预计5-10分钟)  
⚠️ 缺点: 需要安装Node.js

---

```powershell
# 检查生成的文件数量
(Get-ChildItem -Path ".\ELM_JSON" -Filter "*.json" -Recurse).Count

# 验证JSON格式正确性
Get-ChildItem -Path ".\ELM_JSON" -Filter "*.json" -Recurse | Select-Object -First 5 | ForEach-Object {
```

2. **转换失败**: 检查CQL文件语法，查看错误日志
3. **Java版本问题**: 确保使用Java 11或更高版本
