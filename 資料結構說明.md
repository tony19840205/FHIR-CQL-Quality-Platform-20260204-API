### 問題2: FHIR是一個一個病人上傳嗎？

**是的！** 根據你之前的上傳腳本，**不是上傳整個Bundle**，而是**逐一上傳每個資源**。

#### 你之前的上傳方式（正確的）：

```powershell
# 讀取Bundle
$bundle = Get-Content $jsonFile -Raw | ConvertFrom-Json

# 逐一上傳每個資源（不是整個Bundle）
foreach ($entry in $bundle.entry) {
    $resource = $entry.resource
    $resourceType = $resource.resourceType  # Patient, Encounter, etc.
    $resourceId = $resource.id
    
    # 單獨上傳每個資源
    $url = "$baseUrl/$resourceType/$resourceId"
    $body = $resource | ConvertTo-Json -Depth 10
    
    Invoke-RestMethod -Uri $url -Method Put -Body $body -ContentType "application/fhir+json"
}
```

**上傳順序**:
1. PUT Patient/dementia-patient-001
2. PUT Encounter/dementia-encounter-001
3. PUT Condition/dementia-condition-001
4. PUT Patient/dementia-patient-002
5. PUT Encounter/dementia-encounter-002
6. ... 依此類推
