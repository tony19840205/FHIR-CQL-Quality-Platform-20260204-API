# 手動轉換指引 - 5個CQL文件

## 方法：使用 https://cql.dataphoria.org/

### 文件清單：
1. Indicator_03_1_Same_Hospital_Antihypertensive_Overlap_1710.cql
2. Indicator_03_2_Same_Hospital_Lipid_Lowering_Overlap_1711.cql  
3. Indicator_03_3_Same_Hospital_Antidiabetic_Overlap_1712.cql
4. Indicator_03_4_Same_Hospital_Antipsychotic_Overlap_1726.cql
5. Waste.cql

### 步驟：
1. 開啟瀏覽器到 https://cql.dataphoria.org/
2. 對每個文件執行：
   - 打開 cql\[檔名].cql
   - 全選複製 (Ctrl+A, Ctrl+C)
   - 貼到線上編輯器左側 (Ctrl+V)
   - 點擊 "Translate to ELM"
   - 等待右側顯示JSON
   - 全選右側JSON (Ctrl+A)
   - 複製 (Ctrl+C)
   - 新建文件，貼上並保存為 ELM_JSON_OFFICIAL\舊50\[檔名].json

### 或使用輔助腳本：
```powershell
.\AutoConvert_Web.ps1 -CqlFolder "cql" -OutputFolder "ELM_JSON_OFFICIAL\舊50"
```
然後只處理這5個文件即可。
